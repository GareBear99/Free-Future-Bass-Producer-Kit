### How the loops are treated

Every loop uses a **1-bar cosine crossfade** at the loop point:

```
[==== LOOP BODY ====][XFADE TAIL    ][ 300ms silence ]
                     вҶ‘ end fades out
                     вҶ‘ start fades in simultaneously
                     = seamless blend, zero click
```

This means the file is slightly longer than a perfect 2-bar loop вҖ” the extra samples at the end blend back into the beginning. Import as a one-shot or trim to the bar line if you want a perfect loop region in your DAW.

## рҹҺӣпёҸ DAW Usage Tips

**For seamless looping in your DAW:**
1. Drop the loop on a track
2. Set your loop region to exactly 2 bars at the file's BPM
3. The crossfade tail handles the transition automatically вҖ” no need to add your own fade

**For one-shots:**
- All files start at zero amplitude (3ms fade-in applied) so you won't get a click on any trigger

**808s:**
- Tune them in your sampler вҖ” they're pitched to A1, C2, E2, G2
- Layer the sub with your kick for maximum punch
